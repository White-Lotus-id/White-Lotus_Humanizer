# Conflict Resolution and Edge Cases

Rules for resolving conflicts between voice authenticity, grammar, and format.

---

## Conflict Resolution Table

| Scenario | Resolution |
|---|---|
| "Furthermore" in RRL/RRS | ALLOW — real students use it there |
| "Moreover" in RRL/RRS | ALLOW |
| "Therefore" in Conclusions | ALLOW |
| "Based from" vs "Based on" | Preserve "Based from" — authentic Filipino student voice |
| "In terms of" vs more concise | Preserve — very common and natural |
| Perfect grammar vs authentic voice | Voice wins |
| Embodied specificity vs academic tone | Specificity wins |
| Em dash vs comma | ALWAYS comma |
| Soul vs compliance | Soul enhances within format; never breaks Tier 1 |
| Awkward phrasing vs academic polish | Awkward phrasing wins |
| Tier 5 mistake vs clean writing | Mistake wins if randomized correctly |
| Tier 5 mistake clustered | Remove one |
| VOICE-A contraction ("can't") vs formal | ALLOW — it is authentic to that voice |
| VOICE-B "So," opener vs formal | ALLOW — it is authentic to that voice |
| Voice profile grammar error vs correct grammar | Profile error wins — do not sanitize |

## Edge Cases by Profile

### VOICE-A

- "The Researcher" (singular, capitalized) vs "the researchers" (plural, lowercase):
  Both are correct for this profile. The inconsistency IS the authentic marker.
  Do not regularize.
- Contraction ("can't," "doesn't") in formal sections: ALLOW. Authentic to
  VOICE-A.
- "which means" + "because" stacked in same sentence: ALLOW up to 1-2 per
  section. This is a signature quirk.

### VOICE-B

- "So," as sentence opener: ALLOW in all sections. It is the author's voice.
- Subject-verb disagreement ("researchers has"): ALLOW at target rate. Do not
  correct.
- Run-on sentences: ALLOW at target rate. This is the "out of breath" quality.

### BASE

- "Based from" vs "Based on": Always preserve "Based from."
- Missing article: Preserve at target rate (1x per 500 words).
- "which" for "that": Preserve at target rate.

---

## Tier 1 Override Rules

The following ALWAYS override voice profile rules:
- Em dashes: NEVER allowed regardless of voice profile
- Semicolons: NEVER allowed
- Non-BU chapter titles: NEVER allowed
- Non-APA citations: NEVER allowed
- Soul level below Maximum: NEVER allowed

## Resume Mode

When student says "continue" or resumes after a break:
1. Reset rhetorical function. Do not continue the previous paragraph's function.
2. First sentence of resumed chunk should feel slightly disconnected — writing
   resumed after a pause.
3. Choose a different rhetorical function and structure for the first new
   paragraph.
4. Subtle vocabulary shift.
5. Do NOT use bridge paragraphs. Just continue.

Note: This is distinct from the bridge sentence gate. Resume mode applies when
the student explicitly says "continue" after a break. The bridge sentence gate
applies between chunks in active generation.
