# Authenticity Checklist (Phase 4 — Internal Only)

Run after each draft chunk internally. Nothing in this checklist is ever
appended to output or announced to the student.

---

## Anti-AI Pass

Internally ask: "What makes this section still obviously AI-generated?"

### Structural Checks

- [ ] Paper-anchor formula applied correctly for this section and voice profile?
- [ ] Full S1-S5 formula used, not just S1?
- [ ] Same paragraph structure used in consecutive paragraphs? If yes, vary.
- [ ] Same connector density in consecutive paragraphs? If yes, vary.
- [ ] Three consecutive paragraphs with same grammatical opener? If yes, vary.

### Voice Checks

- [ ] Embodied specificity present? (min 1 per 250 words in Intro and Results)
- [ ] Problem genealogy multi-causal? (Introduction only)
- [ ] Sharp punch sentences present? (1 per 5 paragraphs)
- [ ] Spontaneous thought patterns injected at correct frequency?
- [ ] Awkward phrasing and redundancy at natural frequency?
- [ ] ESL voice check: authentic Filipino student or sanitized to "native speaker"?
- [ ] Soul check: does this read like a student who genuinely cares?

### Vocabulary Checks

- [ ] AI vocabulary replaced with plain words? (see Section 2.1 master list)
- [ ] Em dashes removed?
- [ ] Semicolons removed?
- [ ] Negative parallelisms removed?
- [ ] Synonym cycling stopped? (does not swap synonyms to "avoid repetition")
- [ ] Lexical repetition preserved? Main research construct repeats exactly — no synonym substitution (2.15.1)
- [ ] Filipino-specific terms preserved? (brownout, barangay, sari-sari store)

### Naeem Resistance Checks

- [ ] Clunky passive construction used at least once per Research Design / Results? (2.15.2)
- [ ] Mid-sentence citation inserted before grammatical resolution in RRL/RRS? (2.15.3)
- [ ] Tense asymmetry present within at least one paragraph? (2.15.5)
- [ ] Paragraph openers NOT artificially rotated when same topic continues? (2.15.6)

### VOICE-C Authenticity Checks (v6.5.0 — run only when VOICE-C is declared)

- [ ] Nominalisation density: at least one of-chain per long sentence? (2.16.1)
- [ ] ESL marker distribution within density targets per 500 words? (tier5_mistake_layer.md — VOICE-C Density Targets table)
- [ ] Connector rotation: no same sentence-initial connector ("Moreover," / "However," / etc.) twice in a row?
- [ ] Per-hypothesis 5-move template followed in Discussion section? (voice_c.md — Discussion 5-move template)
- [ ] Chapter sandwich pattern present at chapter open and close? ("This chapter…" preview + "Chapter [N] contains…" summary)
- [ ] Construct abbreviations used exclusively after first introduction? (No re-expansion of FIN, TOE, PEX, etc. mid-chapter)
- [ ] British/Commonwealth spelling enforced? (fertiliser, utilised, behaviour, modelling, operationalisation) — no American spellings except preserved inconsistencies ("program")
- [ ] Inline numbered list format used instead of bullet lists? ("1) … 2) … 3) …" in body text)

### Grammar and Mistake Checks

- [ ] Voice profile grammar errors injected at correct rate?
- [ ] Tier 5 mistakes scattered, non-patterned?
- [ ] No cluster of Tier 5 mistakes in same paragraph?
- [ ] No more than one Tier 5 mistake type used twice in a row?
- [ ] Mistake position rotated from previous paragraph?
- [ ] Tier 5 category rotated (Confirmed → Naeem → Rhetorical → Mechanical)?

### Specificity Checks

- [ ] Every abstract claim has a concrete anchor?
- [ ] All numbers are actual numbers, not "high" or "low"?
- [ ] Every component named exactly? ("ESP32" not "a microcontroller")
- [ ] Beneficiaries named specifically? (not "students in general")

## Common AI Tells — Actively Remove

| AI Pattern | Replace With |
|---|---|
| "It is worth noting that" | Drop or "Also," |
| "In conclusion, it is evident that" | "Therefore," + direct claim |
| "This highlights the importance of" | State the importance directly |
| "Furthermore, it should be noted" | "Also," |
| "In light of this" | "Because of that," |
| "This is of great significance" | State the specific significance |
| Perfectly parallel sentence structures | Introduce length and structure variation |
| Every paragraph 3-4 sentences exactly | Mix 2-sentence and 5-sentence paragraphs |
| Balanced "While X... Y" every paragraph | Use other contrast structures |
| Passive voice everywhere | Mix with active, especially for researcher actions |
