# APA 7th Edition Citation Rules

For institutional thesis compliance.

---

## In-Text Citations

- Single author: (Author, Year)
- Two authors: (Author & Author, Year)
- Three or more authors: (Author et al., Year)
- Direct quote: (Author, Year, p. XX)
- Placeholder when no source confirmed: `[CITATION: Author, Year, DOI/URL]`

---

## Reference List Format

- Alphabetical by last name
- Hanging indent
- Single-spaced entries
- Double-spaced between entries (or follow institutional spec if different)
- Placed at end of document under "BIBLIOGRAPHY"
- institutional heading: BIBLIOGRAPHY (all caps, not "References")

---

## Common Reference Formats

### Journal Article
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Journal, volume(issue), pages. https://doi.org/xxxxx
```

### Book
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

### Chapter in Edited Book
```
Author, A. A., & Author, B. B. (Year). Title of chapter. In A. Editor & B. Editor (Eds.), Title of book (pp. xxx-xxx). Publisher.
```

### Website
```
Author, A. A. (Year, Month Day). Title of page. Site Name. URL
```

### Conference Paper
```
Author, A. A. (Year, Month Day-Day). Title of paper. In B. B. Chairperson (Chair), Title of symposium [Paper presentation]. Name of Conference, Location. URL
```

---

## Verification Protocol

For every citation submitted:
1. Web search for abstract
2. Verify claim match
3. Check APA 7th format
4. Flag:
   - `[VERIFIED VIA SEARCH]` — abstract found, claim matches
   - `[UNVERIFIABLE]` — abstract not found in search
   - `[REFERENCE MISMATCH]` — source found but does not support the claim

If search fails: ask student for abstract or DOI. Do not fabricate.

---

## Bibliography Format (the University Specific)

BIBLIOGRAPHY — Alphabetical by last name, APA 7th, hanging indent, single
spaced, at end of document.

---

## Reference Protection Rule

When the user requests a chapter that includes a reference list, OR requests
a reference list alone:
- Output the reference block exactly as APA 7th requires
- Apply ZERO voice layer to reference entries
- Apply ZERO restructuring
- Apply ZERO word substitution
- Reference entries are frozen output — they cannot be humanized
