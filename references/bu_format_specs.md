# Institutional Thesis Format Specifications

Compliance: Institutional Thesis Format

---

## 1.1 Format Specifications

| Element | Spec | Violation Flag |
|---|---|---|
| Margins | 1.5" left; 1" top, bottom, right | `[FORMAT ERROR: Margin]` |
| Font | Times New Roman 12pt | `[FORMAT ERROR: Font]` |
| Spacing | Double (Ch 1-5); Single (Bibliography, Prelims) | `[FORMAT ERROR: Spacing]` |
| Spacing after punctuation | Single space after commas, colons, semicolons | `[FORMAT ERROR: Punctuation spacing]` |
| Alignment | Justified | `[FORMAT ERROR: Alignment]` |
| Pagination | Roman numerals center-bottom (Prelims); Arabic numerals top-right (Main) | `[FORMAT ERROR: Pagination]` |
| Abstract | 250-300 words | `[FORMAT ERROR: Abstract length]` |
| Citations | APA 7th Edition | `[CITATION ERROR: Non-APA]` |
| Table captions | Above table; Times New Roman 12pt, regular, left-aligned | `[FORMAT ERROR: Table caption]` |
| Figure captions | Below figure; Times New Roman 12pt, center-aligned | `[FORMAT ERROR: Figure caption]` |

---

## 1.2 Chapter Titles (Exact)

- Ch 1: THE PROBLEM
- Ch 2: REVIEW OF RELATED LITERATURE AND STUDIES
- Ch 3: RESEARCH DESIGN
- Ch 4: DATA ANALYSIS AND INTERPRETATION
- Ch 5: SUMMARY OF FINDINGS, CONCLUSIONS AND RECOMMENDATIONS
- Bibliography: BIBLIOGRAPHY (alphabetical, APA 7th, by last name)

---

## 1.3 Voice and Person

| Use | Never Use |
|---|---|
| "the researchers" | "we" |
| "the study" | "I," "our" |
| "the researchers found" | "it was found that" (when actor known) |

Passive only when: no actor identifiable OR scientific convention ("data were
analyzed").
Exception: VOICE-A and VOICE-B profiles allow controlled "we" slips — see
voice profile files.

## 1.5 Quantification Mandate

| Avoid | Use |
|---|---|
| "high/low" | [X units] or [X]% vs [baseline] |
| "fast/slow" | [X]ms, [X]fps, [X×] real-time |
| "significant" (stats) | [p-value] or [CI] |
| "improved" | [X]% improvement over [baseline] |

## Voice Rotation Rules (Chapter-Level)

| Chapter / Section | Default Voice Target | Allowed Alternates |
|---|---|---|
| Chapter 1 (all sections) | VOICE-A or VOICE-B | BASE |
| Chapter 2 — RRL | VOICE-A only | none |
| Chapter 2 — RRS | VOICE-A only | none |
| Chapter 2 — Synthesis | VOICE-A only | none |
| Chapter 3 | VOICE-A or VOICE-B | BASE |
| Chapter 4 | VOICE-A or VOICE-B | BASE |
| Chapter 5 | VOICE-A or VOICE-B | BASE |
| Non-thesis requests | VOICE-A or VOICE-B | BASE |

Rule: Do not use the same profile for two consecutive chunks.
